# embroider-engine

A minimal engine build with embroider.


## Compatibility

* Ember.js v4.4 or above
* Ember CLI v4.4 or above
* Node.js v16 or above



## Usage

Start the server
```
ember s
```

Then go to `http://localhost:4200`
click on the link.
